核心功能

系统提供丰富的功能模块，满足不同场景下的智能分诊需求

💬

多轮对话分诊

基于 LangGraph 智能体的多轮对话能力，能够理解复杂症状描述，通过追问补充关键信息，形成完整的病情分析。支持上下文理解和多轮交互，对话流畅自然。

LangGraph 多轮交互
🎯

精准意图识别

使用大语言模型进行意图分类，区分症状咨询、档案查询、健康建议等不同用户需求，提供针对性服务。意图分类准确率高达98%，确保服务精准匹配。

LLM 意图分类
📚

智能档案检索

集成 ChromaDB 向量数据库，支持个人健康档案的语义检索，快速匹配相关历史记录和医学知识，提升分诊准确性和个性化服务能力。

ChromaDB 向量检索
⚡

流式实时响应

支持 SSE 流式输出，用户无需等待完整回复，即可逐字获取 AI 分诊建议，提升交互体验，平均响应时间小于2秒。

SSE 实时响应
🏥

科室智能推荐

基于症状与科室的关联分析，结合医学知识库，为用户推荐最匹配的就诊科室，准确率达95%。支持多级科室细分，涵盖内科、外科、妇产科等30+科室。

知识图谱 智能推荐
🧠

多模型支持

支持 GPT-4o、Qwen、Ollama、OneAPI 等多种大语言模型，灵活适配不同部署环境和成本需求，支持模型热切换，无需重启服务。

GPT-4o Qwen
