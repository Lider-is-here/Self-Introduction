项目背景

为什么做这个项目？

随着人们生活水平的提高，越来越多的人开始在家里养花种草。然而，很多人由于缺乏专业的养护知识，经常导致花卉枯萎死亡。

同时，传统花店的销售模式难以满足现代人的购物需求，线上线下融合成为趋势。物联网技术的发展也为智能花盆的普及提供了技术基础。

基于以上痛点，我们开发了"植管家"智能花卉管理小程序，旨在帮助用户科学养护花卉，同时提供便捷的鲜花购买和社区交流服务。

前端技术

微信小程序 WXML WXSS JavaScript ES6+ 自定义组件 Canvas 图表 地理位置 API 网络请求封装

核心技术亮点

  • 自定义导航栏组件，实现全局统一的顶部导航
  • 数据请求统一封装，支持 baseUrl 配置和错误处理
  • 图片懒加载、错误兜底、HTTPS 协议转换
  • 完整的页面生命周期管理，数据状态同步

后端技术

Java Spring Boot MyBatis MySQL 阿里云 OSS Swagger RESTful API MQTT

核心技术亮点

  • Spring Boot 2.x 框架，自动配置，快速开发
  • MyBatis 持久层，XML + 注解混合使用
  • 阿里云 OSS 图片存储，支持图片上传和 CDN 加速
